Book class reference

  • bool load(const wchar_t* filename, const wchar_t* tempFile = 0)
    将整个文件加载到内存中。指定一个临时文件以减少内存消耗。如果发生错误，返回 false。使用Book::errorMessage()获取错误信息。
  • bool loadSheet(const wchar_t* filename, int sheetIndex, const wchar_t* tempFile = 0)
    只将文件指定索引的工作表加载到内存中。指定一个临时文件以减少内存消耗。如果发生错误，返回 false。使用Book::errorMessage()获取错误信息。
  • bool loadPartially(const wchar_t* filename, int sheetIndex, int firstRow, int lastRow, const wchar_t* tempFile = 0)
    只将文件指定索引的工作表和行范围载入内存。指定一个临时文件以减少内存消耗。如果发生错误，返回 false。使用Book::errorMessage()获取错误信息。
  • bool save(const wchar_t* filename, bool useTempFile = false)
    将当前工作簿保存到文件中。使用临时文件以减少内存消耗。如果发生错误，返回 false。使用Book::errorMessage()获取错误信息。
  • bool loadInfo(const wchar_t* filename)
    只加载工作表的信息。之后你可以调用 Book::sheetCount() 和 Book::getSheetName() 方法。 如果发生错误，返回 false。使用Book::errorMessage()获取错误信息。
  • bool loadRaw(const char* data, unsigned size, int sheetIndex = -1, int firstRow = -1, int lastRow = -1)
    从用户的内存缓冲区加载文件。参数：
    data - 缓冲区指针；
    size - 缓冲区中数据的大小；
    sheetIndex - 只加载文件中指定索引的工作表，-1加载所有工作表；
    firstRow - 加载范围的首行，-1加载所有行，直到 lastRow；
    lastRow - 加载范围的尾行，-1加载 firstRow 之后的所有行。
    如果发生错误，返回 false。使用Book::errorMessage()获取错误信息。
  • bool saveRaw(const char** data, unsigned* size)
    保存文件到内部内存缓冲区。参数：
    data - 缓冲区指针；
    size - 保存大小
    如果发生错误，返回 false。使用Book::errorMessage()获取错误信息。
  • Sheet* addSheet(const wchar_t* name, Sheet* initSheet = 0)
    添加一个新的 sheet 到工作簿， 返回工作表指针。如果要复制现有的工作表，请使用 initSheet 参数。注意 initSheet 必须只来自于本工作簿。工作表名称中不能使用以下字符：
    - 冒号(:)
    - 反斜杠(\)
    - 星号(*)
    - 问号(?)
    - 斜杠(/)
    - 开方括号([)
    - 闭方括号(])
    工作表名称不能以单引号（'）字符开头或结尾。 如果发生错误，返回 NULL。使用Book::errorMessage()获取错误信息。
  • Sheet* insertSheet(int index, const wchar_t* name, Sheet* initSheet = 0)
    在 index 位置插入一个新的 sheet 到工作簿，返回工作表指针。如果你想复制一个现有的工作表，请使用 initSheet 参数。注意 initSheet 必须只来本工作簿。如果发生错误，返回NULL。使用Book::errorMessage()获取错误信息。
  • Sheet* getSheet(int index) const
    获取指向 sheet 的指针。如果使用 Book::loadSheet() 或 Book::loadPartially() 方法，使用索引应小于 Book::sheetCount() 方法的返回值。如果发生错误，返回NULL。使用 Book::errorMessage()方法取得错误信息。
  • const wchar_t* getSheetName(int index) const
    返回指定索引的工作表名称。如果发生错误则返回 NULL。使用Book::errorMessage()获取错误信息。
  • SheetType sheetType(int index) const
    返回指定索引的工作表类型。
    工作表类型值描述
    SHEETTYPE_SHEET标准工作表
    SHEETTYPE_CHART图表工作表
    SHEETTYPE_UNKNOWN未知工作表
  • bool moveSheet(int srcIndex, int dstIndex)
    将索引 srcIndex 的工作表移动到索引 dstIndex 的工作表前面。如果发生错误，返回 false。使用Book::errorMessage()获取错误信息。
  • bool delSheet(int index)
    删除指定索引的 sheet。如果发生错误，返回 false。使用 Book::errorMessage()获取错误信息。
  • int sheetCount() const
    返回工作簿中工作表的数目。
  • Format* addFormat(Format* initFormat = 0)
    添加新的 format 到工作簿，初始参数可以从其他 format 复制。如果发生错误，返回 NULL。请注意，在载入文件后，它会被移除。
  • Font* addFont(Font* initFont = 0)
    添加新的 font 到工作簿，初始参数可以从其他 font 复制。如果发生错误，返回 NULL。
  • RichString* addRichString()
    在工作簿中添加一个新的 rich string，以便使用 Sheet::writeRichStr() 方法在一个单元格中使用不同的字体。不要手动释放返回指针
  • int addCustomNumFormat(const wchar_t* customNumFormat)
    向工作簿添加新的自定义数字格式。格式字符串 customNumFormat 表示如何格式化和呈现单元格的数值。请参阅 自定义格式字符串指南。 返回自定义格式标识符。它将被 Format::setNumFormat() 使用。如果发生错误，返回 0。使用 Book::errorMessage()获取错误信息。
  • const wchar_t* customNumFormat(int fmt) const
    返回指定自定义格式标识符 fmt 的自定义格式字符串。参见自定义格式字符串指南。如果发生错误，返回 NULL。使用 Book::errorMessage()获取错误信息。
  • Format* format(int index)
    返回一个带有定义索引的format方法。索引必须小于 formatSize() 方法的返回值。
  • int formatSize()
    返回工作簿中格式大小。
  • Font* font(int index)
    返回一个带有定义索引的 font 方法。索引必须小于 fontSize() 方法的返回值。
  • int fontSize()
    返回工作簿中字体大小。
  • double datePack(int year, int month, int day, int hour = 0, int min = 0, int sec = 0, int msec = 0)
    将日期和时间信息打包成 double 类型。
  • bool dateUnpack(double value, int* year, int* month, int* day,
    int* hour = 0, int* min = 0, int* sec = 0, int* msec = 0)
    解包 double 类型的日期和时间信息。如果发生错误，返回 false。
  • Color colorPack(int red, int green, int blue)
    打包红、绿、蓝三原色到 color 类型。
  • void colorUnpack(Color color, int* red, int* green, int* blue)
    解包 color 类型到红、绿、蓝三原色
  • int activeSheet() const
    返回此工作簿中的活动工作表索引。
  • void setActiveSheet(int index)
    设置此工作簿中的活动工作表索引。
  • int pictureSize() const
    返回本工作簿中的图片数量。
  • PictureType getPicture(int index, const char** data, unsigned* size) const
    返回内存缓冲区中 index 位置的图片。参数：
    index - 在工作簿中的位置；
    data - 对缓冲区的引用；
    size - 对保存的尺寸的引用。
    返回图片的类型：
    图片类型值描述
    PICTURETYPE_PNGPNG 格式
    PICTURETYPE_JPEGJPEG 格式
    PICTURETYPE_GIFGIF 格式
    PICTURETYPE_WMFWMF 格式
    PICTURETYPE_DIBDIB 格式
    PICTURETYPE_EMFEMF 格式
    PICTURETYPE_TIFFTIFF 格式
    PICTURETYPE_ERROR使用 Book::errorMessage() 获取错误信息。
  • int addPicture(const wchar_t* filename)
    添加一张图片到工作簿，返回一个图片标识符。支持BMP, DIB, PNG, JPG和WMF图片格式。图片标识符供 Sheet::setPicture() 使用。如果发生错误，返回 -1。使用 Book::errorMessage()获取错误信息。
  • int addPicture2(const char* data, unsigned size)
    从内存缓冲区向工作簿添加图片：
    data - 指向缓冲区的指针，包含图片数据（BMP、DIB、PNG、JPG或WMF格式）；
    size - 缓冲区中数据的大小。 返回一个图片标识符。图片标识符供Sheet::setPicture() 使用。如果发生错误，返回 -1。使用 Book::errorMessage()获取错误信息。
  • int addPictureAsLink(const wchar_t* filename, bool insert = false)
    将图片作为链接添加到工作簿中（仅适用于xlsx文件）：
    insert = false - 只存储一个文件的链接；
    insert = true - 存储一张图片和一个文件的链接。
    返回一个图片标识符。支持BMP, DIB, PNG, JPG和WMF图片格式。图片标识符供 Sheet::setPicture() 使用。如果发生错误，返回 -1。使用 Book::errorMessage() 获取错误信息。
  • const wchar_t* defaultFont(int* fontSize)
    返回此工作簿的默认字体名称和大小。如果发生错误则返回 0。
    使用 Book::errorMessage() 获取错误信息。
  • void setDefaultFont(const wchar_t* fontName, int fontSize)
    设置此工作簿的默认字体名称和大小。
  • bool refR1C1() const
    返回R1C1参考模式是否处于激活状态。
  • void setRefR1C1(bool refR1C1 = true)
    设置R1C1参考模式。
  • bool rgbMode()
    返回RGB模式是否处于激活状态。
  • void setRgbMode(bool rgbMode = true)
    设置RGB模式：
    true - RGB模式；
    false - Index 模式（默认）。
    在RGB模式下，使用 colorPack() 和 colorUnpack() 方法来获取/设置颜色。
  • CalcModeType calcMode() const
    返回工作簿的计算模式。
  • void setCalcMode(CalcModeType calcMode)
    设置工作簿的计算模式：
    计算模式值描述
    CALCMODE_MANUAL手动计算
    CALCMODE_AUTO自动计算
    CALCMODE_AUTONOTABLE除数据表外自动
  • int biffVersion() const
    返回二进制文件的BIFF版本，仅用于xls格式。
  • bool isDate1904() const
    返回 1904年的日期系统是否处于激活状态。
  • void setDate1904(bool date1904 = true)
    设置日期系统模式：
    true - 1904年日期系统；
    false - 1900年日期系统（默认）。
    在1900年的日期基数系统中，下限为1900年1月1日，其序列值为 1。
    在1904年的日期基数系统中，下限为1904年1月1日，其序列值为 0。
  • bool isTemplate() const
    返回工作簿是否为模板。
  • void setTemplate(bool tmpl = true)
    设置模板标志：
    true - 工作簿是模板；
    false - 工作簿不是模板（默认）。
    它允许将文件类型从模板文件（xlt和xltx）改为常规文件（xls和xlsx），反之亦然。
  • void setKey(const wchar_t* name, const wchar_t* key)
    设置客户的许可证密钥。
  • bool setLocale(const char* locale)
    设置该库的语言环境。
    locale 参数与 C 运行时库中的 setlocale() 函数中的 locale 参数相同。
    例如，值 "en_US.UTF-8 " 允许在 Linux 或 Mac 中使用非 ascii字符。
    它接受特殊值 "UTF-8"，以便在 Windows 和其他操作系统中使用 UTF-8 字符编码。
    它对具有宽字符串的 unicode 项目没有影响（使用 _UNICODE 预处理器变量）。
    如果给定了一个有效的 locale 参数，则返回 true。
  • const char* errorMessage() const
    返回最后一条错误信息。
  • void release()
    删除此对象并释放资源。
