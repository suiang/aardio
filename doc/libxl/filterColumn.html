FilterColumn class reference

  • int index() const
    返回该自动过滤列的零基索引。
  • Filter filterType() const
    返回该自动过滤列的过滤类型。
    过滤值描述
    FILTER_VALUE按指定值过滤
    FILTER_TOP10要过滤的前N名（项目的百分比或数量）
    FILTER_CUSTOM用指定的标准自定义过滤器
    FILTER_DYNAMIC指定动态过滤条件
    FILTER_COLOR指定要过滤的颜色
    FILTER_ICON指定要过滤的图标
    FILTER_EXT定义灵活的存储扩展功能
    FILTER_NOT_SET没有过滤器
  • int filterSize() const
    ‎返回过滤器值的数量。
  • const wchar_t* filter(int index) const
    根据索引返回过滤器的值。
  • void addFilter(const wchar_t* value)
    添加过滤器的值。
  • bool getTop10(double* value, bool* top, bool* percent)
    获取顶部或底部项目的数量：
    value - 项目数量；
    top - 如果为真，则为最上面的项目，否则为最下面的项目；
    percent - 使用百分比代替数字项。
    如果错误，返回 false。使用 Book::errorMessage() 获取错误信息。
  • void setTop10(double value, bool top = true, bool percent = false)
    设置顶部或底部项目的数量：
    value - 项目数量；
    top - 如果为真，则为最上面的项目，否则为最下面的项目；
    percent - 使用百分比代替数字项。
  • bool getCustomFilter(Operator* op1, const wchar_t** v1, Operator* op2, const wchar_t** v2, bool* andOp)
    获取自定义的过滤条件：
    op1 - 第一个过滤标准中的过滤比较所使用的操作符；
    v1 - 第一个过滤标准中使用的值；
    op2 - 第二个过滤标准中的过滤比较所使用的操作符；
    v2 - 第二个过滤标准中使用的值；
    andOp - 指示两个标准是否有 "和 "的关系的标志。true 表示 "and"，false表示 "or"。
    如果错误，返回 false。使用 Book::errorMessage() 获取错误信息。
  • void setCustomFilter(Operator op1, const wchar_t* v1, Operator op2 = OPERATOR_EQUAL, const wchar_t* v2 = 0, bool andOp = false)
    设置自定义过滤条件：
    op1 - 第一个过滤标准中的过滤比较所使用的操作符；
    v1 - 第一个过滤标准中使用的值；
    op2 - 第二个过滤标准中的过滤比较所使用的操作符；
    v2 - 第二个过滤标准中使用的值；
    andOp - 标志，表示两个标准是否有 "and" 的关系。True 表示 "and"，false表示 "or"。
    运算符值描述
    OPERATOR_EQUAL=
    OPERATOR_GREATER_THAN>
    OPERATOR_GREATER_THAN_OR_EQUAL>=
    OPERATOR_LESS_THAN<
    OPERATOR_LESS_THAN_OR_EQUAL<=
    OPERATOR_NOT_EQUAL<>
  • void clear()
    清除过滤条件。
