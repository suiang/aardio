    使用 Book::errorMessage() 获取错误信息。

    如果发生错误，返回 false。使用 Book::errorMessage() 获取错误信息。

    使用 Book::errorMessage() 方法获取错误信息。

  • bool setHeader(const wchar_t* header, double margin = 0.5)
    设置打印时页面的页眉文字。打印时，文字会出现在每页的顶部。文本的长度必须小于或等于 255。
    页眉文本可以包含特殊命令，例如页码的占位符、当前日期或文本格式属性。
    特殊命令用单个字母和前导安培符（"&"）表示。边距以英寸为单位。
    代码描述
    &L指定左边部分的开始
    &P指定当前页码
    &N指定总页数
    &\d{1,3}指定文本字体大小，其中字体大小以点为单位，例如：&9 或 &36
    &S指定是否开启或关闭删除线文本样式。
    &X指定上标文本样式是开启还是关闭
    &Y指定下标文本样式是开启还是关闭
    &C指定中心部分的开始
    &D指定一个日期
    &T指定一个时间
    &U指定单下划线文本样式是开启还是关闭
    &E指定双下划线文本样式是开启还是关闭
    &R指定右边部分的开始
    &Z指定工作簿文件路径

    为指定的范围添加数据验证（仅适用于xlsx文件）。

    type - 数据验证的类型：
    数据验证的类型值描述
    VALIDATION_TYPE_NONE没有数据验证
    VALIDATION_TYPE_WHOLE检查满足给定条件的整数值的数据验证
    VALIDATION_TYPE_DECIMAL数据验证，检查满足给定条件的小数值
    VALIDATION_TYPE_LIST数据验证，检查是否有一个值与列表中的一个值相匹配
    VALIDATION_TYPE_DATE数据验证，检查满足给定条件的日期值
    VALIDATION_TYPE_TIME数据验证，检查满足给定条件的时间值
    VALIDATION_TYPE_TEXTLENGTH数据验证，检查文本值，其长度满足给定条件
    VALIDATION_TYPE_CUSTOM使用自定义公式检查单元格值的数据验证

    op - 数据验证的关系运算符：
    DataValidationOperator valueDescription
    VALIDATION_OP_BETWEEN数据验证，检查一个值是否在两个其他值之间
    VALIDATION_OP_NOTBETWEEN数据验证，检查一个值是否不在两个其他值之间
    VALIDATION_OP_EQUAL数据验证，检查一个值是否等于一个指定的值
    VALIDATION_OP_NOTEQUAL数据验证，检查一个值是否不等于一个指定的值
    VALIDATION_OP_LESSTHAN数据验证，检查一个值是否小于一个指定的值
    VALIDATION_OP_LESSTHANOREQUAL数据验证，检查一个值是否小于或等于一个指定的值
    VALIDATION_OP_GREATERTHAN数据验证，检查一个值是否大于指定值
    VALIDATION_OP_GREATERTHANOREQUAL数据验证，检查一个值是否大于或等于一个指定的值

    rowFirst - 范围内的第一行；
    rowLast - 范围内的最后一行；
    colFirst - 范围内的第一列；
    colLast - 范围的最后一列；
    value1 - 关系运算符的第一个值；
    如果要直接指定一个值的列表(例如 "A,B,C")，请使用双引号，
    如果要指定一个带值的区域引用 (例如A1:A6)，请不要使用引号；
    value2 - VALIDATION_OP_BETWEEN 或 VALIDATION_OP_NOTBETWEEN运算符的第二个值；
    allowBlank - 一个布尔值，表示数据验证是否将空或空白条目视为有效，'true' 表示空条目是确定的，并且不违反验证约束；
    allowBlank - 一个布尔值，表示数据验证是否将空或空白条目视为有效；
    hideDropDown - 表示是否显示列表类型的下拉组合框的布尔值。 数据验证(VALIDATION_TYPE_LIST)；
    showInputMessage - 表示是否显示输入提示信息的布尔值；
    showErrorMessage - 一个布尔值，表示当输入无效值时，是否根据指定的标准显示错误警告消息；
    prompt - 输入提示的信息文本；
    errorTitle - 错误提示的标题栏文本；
