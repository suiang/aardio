RichString class reference

  • Font* addFont(Font* initFont = 0)
    为 RichString 添加一个新的 font，供 RichString::addText() 方法使用，初始参数可以从其他 font 复制。
  • void addText(const wchar_t* text, Font* font = 0)
    将一段带有指定 font 的文本(run) 添加到 RichString 中，以便在一个单元格中混合不同的字体。
  • const wchar_t* getText(int index, Font** font = 0)
    从 RichString 中返回一段文本（run）和 font 的指定 index。
  • int textSize() const
    返回 RichString 中文本（run）的大小。
